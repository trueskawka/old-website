RC - day 13

Achievements:

  • self check-in
  • keeping the meditation habit streak
  • warm-ups in Ruby
  • first board-game design session with Tanoy
  • failed shopping turned into a successful trip

Today I've learned how post-it notes and a few hours can be used for an awesome afternoon. I've finally had my first board-game design session with Tanoy.

The game theme is sailing in a small boat through open and dangerous seas. First we did some brainstorming, while munching cookies, and after we've been happy with the initial ideas we got down to prototyping. Making the tiny boats, board and all kinds of tokens made it suddenly very real.

The main takeout from today is that we need to play the game and adjust, rather than just talk about concepts and get more and more excited. Next session will probably happen next week and, if everything goes well, we'll soon create prototypes to play them with other RCers.

This game is supposed to be mainly a case study for game design, so I can learn to do it better. But if something cute and fun happens on the way, then it's even more worth it.

As for the failed shopping - I wanted to buy shoes, as I've managed to break a pair during yesterday's karaoke. I ended up getting a big mug and two books.
