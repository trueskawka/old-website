RC - day 63

Achievements:

  • my pull request for Zulip was merged
  • finished writing initial Outreachy proposal for Zulip
  • refactored my workflow

My pull request for Zulip stream filtering in subscriptions page was merged today!

It was the last weekend before Outreachy application deadline and I’ve managed to finish my initial proposal and submit the application on time. I think it will change a lot over the next three weeks, since I seem to have a lot of ideas and awareness of various needs, but need to construct smarter and more tangible goals for introducing new features.

With some alone time on my hands, I’ve tweaked and refactored my workflow, after reading:

  • Rodrigo’s post on having time on your side
  • an old FreeCodeCamp article on finding time to become a better developer

Furthermore I’ve found a lot of inspiration while reading Dan’s list of programming books you might consider reading, or, as I would call it, a list of programming books you might be so inclined as to potentially try to find a little bit, but not too much, time for, if and, only if, you are so inclined to do.
