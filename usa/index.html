RC - day 47

Achievements:

  • took part in Wireshark workshop
  • signed up for entrepreneurship workshop
  • learned to communicate with RC API
  • planned an app for the community
  • learned to build apps in Sinatra and deploy them to Heroku
  • build a to-do list app

I didn’t suspect that becoming a full-time night-owl can be so satisfying and at the same time so completely out of sync with everyone. I sleep and rest better, but anything happening before noon is out of my reach.

The RC part of the day started late, with some much needed personal errands that took priority over interview prep, watercolours and blog meeting. Thankfully, at least I’ve made it to Mike’s second Wireshark workshop and was able to sign-up for Paul’s intense workshop on entrepreneurship next Saturday.

I wanted to simply refactor one of my old projects, but got inspired to come up with an app for RC community - a pop quiz to help learn fellow Recursers’ names. In order to do that with a minimum setup, I’ve decided to use Ruby and Sinatra, which I first had to learn.

A few hours, one pretty good tutorial and some hundred Google queries later, I’ve created and deployed a simple to-do list app on Heroku. At the moment there are no users, so I’m not taking responsibility for any content that may or may not be added by any third parties. Over the weekend I’ll add some users and authorisation, probably using the RC OAuth, just to practice before the bigger project.

In case anyone wants to try the tutorial (the app is 95% the tutorial verbatim with some CSS flavours on top), it took me some time to figure out how to deploy it to Heroku. Most helpful tips:

  • an empty postgresql database must be initiated:

    heroku addons:add heroku-postgresql:dev

  • migratons happen in heroku console:

    heroku run console
require 'sinatra'
require 'slim'
require 'data_mapper'
require './main'
DataMapper.setup(:default, ENV['DATABASE_URL'] ||
 "sqlite3://#{Dir.pwd}/development.db")  
DataMapper.auto_migrate!

RC - day 46

Achievements:

  • had a presentation about quitting
  • seen community presentations
  • took part in game night
  • refactored an old project

We officially outdid ourselves - got so lost in work, at some point we’ve realized that it’s already 6am and we should get some breakfast. My first time at a diner consisted of three huge French toasts with syrup, something to keep one full for the next 10 hours. After a quick trip to the apartment, I got back to RC to work on my Outreachy application and presentation. I’ll try to write a longer post on the latter, as soon as I figure out what’s the best way of rendering mathematical formulas in HTML.

The community presentations today were fun, some highlights:

  • Fenimore’s Music Server for streaming albums over local network (also, his music preferences seem interesting enough to dig into it just for the sake of listening to something new)
  • Harold’s EpubPress Chrome extension for turning web content into ebooks
  • Noah’s Tiny Dungeon Twitter Bot and what’s even more amazing - someone was inspired to create a Dungeon Player out of the tweets

The game night was pretty intense - there were so many new people, including a lot of alumni, that after playing a short game of Codenames and my first Go game ever (thank you Danielle!), I grabbed a pizza slice and beer to work quietly in a side room. This week has been a bit overwhelming and I felt like working on refactoring some old projects.

I’ve managed to work mostly on one of them, a tiny weather app. I first fixed the old implementation and went on to work on the new one for about two hours. I’ve encountered some little bugs, two that might be useful for future reference:

  • Chrome doesn’t accept anymore geolocation over insecure connections - easily fixed with https
  • the OpenWeatherMap.org has a secure API as a premium feature - after reading up about possible solutions (e.g. getting data from other sources, IP-based), I’ve decided to simply use crossorigin.me

I also wanted to learn something new and decided animating SVG sounds fun enough. All in all, with a few more projects to refactor, I feel pretty comfortable with tackling them over the next week or so.
